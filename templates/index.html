¿Qué es Control Host?

Control Host es una aplicación web que hice durante las vacaciones de Navidad para la asignatura de Lenguaje de Marcas. Es mi primer proyecto serio combinando HTML, CSS y un framework de backend.

He usado Flask (un framework de Python) para el backend porque me pareció más sencillo de aprender para empezar. Para el frontend usé Bootstrap 5 para practicar el uso de frameworks CSS y que todo quede responsive.

Aún hay muchas cosas que mejorar y funciones que añadir, pero el tiempo es limitado.

Características del Proyecto
